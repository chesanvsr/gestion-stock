# Sistema de Gestión de Stock

Aplicación fullstack desarrollada para el control y ajuste de stock de productos.

## 🚀 Funcionalidades

* Alta, baja y modificación de productos
* Ajuste de stock
* Visualización de productos
* Gestión mediante API REST

## 🧠 Tecnologías

### Backend

* Node.js
* Express

### Frontend

* React

## ⚙️ Ejecución

### Backend

cd backend
npm install
npm run dev

### Frontend

cd frontend
npm install
npm run dev
